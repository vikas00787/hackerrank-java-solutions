# Plus Minus Problem (Java)

This project contains the Java solution for the **Plus Minus problem**.

## Problem Statement
Given an array of integers, calculate the ratios of its elements that are:
- Positive
- Negative
- Zero

Print the decimal value of each fraction on a new line with **6 digits after the decimal**.

## Example

Input:
6  
-4 3 -9 0 4 1  

Output:
0.500000  
0.333333  
0.166667  

## Explanation
- Positive numbers = 3
- Negative numbers = 2
- Zeros = 1

Ratios are calculated based on the total number of elements in the array.
